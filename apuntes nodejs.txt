**PRIMER EXAMEN**

¿Qué es Node.js?
Node js es un entorno de programación que utiliza el motor de Google Chrome V8.
Utiliza JavaScript

Características de Node.js:
- No bloqueante
- Asíncrono
- Concurrente
- Single Thread (Event loops)
- Basado en callbacks
- Orientado a eventos

**COSAS PARA EL SEGUNDO EXAMEN**

Cómo funciona Node.js

Usamos su core(núcleo) que contiene diferentes módulos:
- Global
- Process
- Buffer/Stream -> Conexión, carga y descarga de archivos divididos en "pedazos" llamados chunks.
- FileStream -> Uso de ficheros
- Path
- HTTP 
- require(exports)
- Events

¿Qué es NPM?
Un gestor de paquetes para instalar módulos y ampliar la funcionalidad de Node.js
